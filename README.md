# Chat App
## How to install
<p>Fork repository and clone it</p>
<pre>https://github.com/ibrahimdenizz/chat-app.git</pre>
<p>Install dependencies</p>
<pre>
npm install
cd client/ && npm install
</pre>

## Run the app
<p>For server</p>
<pre>npm run server</pre>
<p>For client</p>
<pre>npm run client</pre>
